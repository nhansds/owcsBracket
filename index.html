Bracket rules

The OWCS 2025 World Finals use a 12-team double-elimination bracket. Regional top seeds start directly in round two, opening matches are played First to 2 , the rest are First to 3 except the Grand Final, which is played First to 4.

Click any team to pick a winner, then set the score. Every result automatically propagates to the next matches.
